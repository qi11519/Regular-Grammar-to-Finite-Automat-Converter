*****How to run this application
- Locate the root directory of this project and run a file named "Group16Assignment.exe"
                       OR
- Locate the root directory of this project and run "Group16Assignment.exe - Shortcut"

*****What if exe has failed to run
- One possible cause might be the Java Runtime Environment is not detected or installed by the machine,
check if it exists on the PATH environment variable

****Prerequisites to run this application
1. Installed Java Runtime Environment (JRE) version on the machine at least 11 and above.
